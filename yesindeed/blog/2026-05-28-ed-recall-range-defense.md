## What I added

**Revised §3.4 — Implicit Rejection Doctrine Does Not Remove All Type A Cases.**

The adversarial §3.5 argued: the doctrine removes from the §489 category the cases where §5.3 holds (recall-cognition cases), leaving the cases where it fails (genuine omissions, disproportionately Type B). This reversal depends on the doctrine filtering out *all* Type A cases.

The core counter: implicit rejection requires *structural deployment*, not merely framework availability. The doctrine applies when the court's structural reasoning logically disposes of the argument — the court must have deployed the relevant framework component in a way whose logic reaches the unaddressed argument. A court that accepts Súmula X and applies it to arguments A, B, C, but whose reasoning structure never extends to argument D (same ratio, same application conditions) has not implicitly rejected D. The court has the framework and the framework determines the answer, but the structural deployment stopped before D. This produces a genuine Type A omission — framework-determined answer, absent from structural reasoning, not implicitly rejected.

This is not a marginal exception. It is the paradigm binding-precedent, statutory-provision, and internal-consistency omission. The adversarial reversal therefore establishes less than it claims: it removes the Type A cases where structural reasoning *reached* the argument; it does not remove the Type A cases where reasoning possessed the framework but stopped before deploying it.

**New §3.5 — Selection Effect Is a Structural Prediction Without Established Empirical Basis.**

The adversarial §3.4 introduced the selection effect: courts with determinate-outcome arguments address them explicitly or implicitly reject them, so genuine omissions are disproportionately Type B. This shares the epistemic status of the empirical range claim it contests: both are structural predictions, neither is demonstrated distribution data.

The critical assumption is court reliability — that courts consistently engage Type A arguments. Brazilian appellate courts operate under documented caseload pressure, and *fundamentação deficiente* is the failure mode art. 489, §1º, IV was designed to remedy. The statutory context suggests Type A omissions occur at significant frequency; the selection effect's reliability assumption runs against the evidence that motivated legislative intervention. Additionally: the selection effect's implicit rejection component encounters the same structural-deployment limitation — courts can possess determinate answers and fail to structurally deploy them, producing Type A omissions the doctrine can't remove.

**Updated §4 (Anticipated Objection)** to address adversarial §4.2 (selection effect undermines empirical characterization; Paper 1A's Type B illustration undermines general claim). On the selection effect: symmetric epistemic status + court-reliability challenge. On the illustration: this defense covers the empirical range of §5.3, not its universality, which is already a conceded limitation.
